# Design Guidelines: Personal Introduction Website - Hong Seong-hoon (POSCO)

## Design Approach
**Reference-Based Approach**: Drawing inspiration from premium minimalist portfolios like Linear, Notion, and modern B2B platforms. Emphasizes sophisticated restraint with Korean corporate excellence through monochromatic elegance and subtle hierarchy.

## Core Design Elements

### Color Palette
**Primary Colors:**
- Pure White: 0 0% 100% (primary background)
- Light Gray: 0 0% 96% (section backgrounds, subtle divisions)
- Medium Gray: 0 0% 45% (secondary text, descriptions)
- Charcoal: 0 0% 25% (primary text, important elements)
- Deep Black: 0 0% 8% (headlines, strong emphasis, navigation)

**Accent Colors:**
- Steel Gray: 210 8% 65% (hover states, borders)
- Soft Gray: 0 0% 85% (card backgrounds, form fields)

### Typography
- **Primary Font**: Inter via Google Fonts (excellent Korean support)
- **Korean Font**: Noto Sans KR for Korean text elements
- **Hierarchy**: Bold black headlines (2xl-4xl), charcoal body text (base-lg), medium gray secondary text, generous line spacing (1.7-1.9)

### Layout System
**Tailwind Spacing Primitives**: Units of 4, 8, 12, 16
- Generous section padding: p-12, p-16, p-20
- Comfortable component margins: m-8, m-12
- Element spacing: gap-8, gap-12, gap-16
- Card spacing: p-6, p-8, p-10

### Component Library
**Core Elements:**
- Clean navigation with white backdrop and black text
- Minimalist card components with soft gray backgrounds
- Forms with charcoal labels and light gray fields
- Timeline elements with black and steel gray progression
- Grid layouts with subtle gray hover effects
- Refined transitions emphasizing restraint

**Visual Treatment:**
- Subtle gray shadows (shadow-sm, shadow-md)
- Minimal rounded corners (rounded-md, rounded-lg)
- Clean borders in steel gray
- Monochromatic gradient overlays where needed

### Content Sections (Maximum 4 sections)
1. **Hero Section**: Professional introduction with large portrait and clean typography hierarchy
2. **Professional Journey**: POSCO career timeline with milestone achievements
3. **Expertise & Skills**: Clean grid showcasing technical and professional capabilities
4. **Contact**: Minimalist engagement section with professional contact methods

## Images
- **Large Hero Image**: Professional corporate portrait in business attire, black and white or desaturated treatment, positioned prominently in hero section
- **Background Elements**: Subtle geometric patterns in light gray, minimal abstract corporate imagery
- **Professional Graphics**: Clean line illustrations in charcoal representing industrial excellence and growth
- **Supporting Visuals**: Optional workplace environment photos with monochromatic treatment

## Key Design Principles
- **Sophisticated Minimalism**: Restrained color palette creating professional authority without coldness
- **Korean Corporate Excellence**: Clean hierarchy respecting traditional business presentation with modern minimalist approach
- **Refined Contrast**: Strategic use of black for emphasis against gray foundations
- **Professional Restraint**: Elegant simplicity that conveys competence and reliability through sophisticated design choices
